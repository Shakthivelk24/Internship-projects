# Full-Stack Development Project  
## Blogging Platform

## Introduction  
This project is a complete blogging platform where users can create, read, update, and delete blog posts with paging, search, and comment support. The client side is built using React, while the server side uses Node.js with Express and MongoDB to store and manage data.

## Project Objectives  
- Build React pages to list, view, and edit blog posts 
- Create backend CRUD APIs to manage posts and comments  
- Add search and paging logic to load data smoothly
- Add login guard to protect edit and delete routes 
- Design data models to store posts and comments 

## Expected Deliverables  
- Blog user interface in React  
- CRUD APIs on the server side  
- Search and paging in post listing
- Login system to guard edit actions 

## Evaluation Criteria  
- CRUD works as planned  
- Login is needed to edit or remove posts  
- Responsive user interface 
- Client side linked well with server side

## Tech Stack  
- React  
- Node.js  
- Express  
- MongoDB  
- JSON Web Token (JWT)  
- bcrypt  
- Axios
- React Router

## Project Setup 

### Server Setup


### Client Setup

## Features
- Create, edit, view, and remove blog posts
- Add and view comments on posts  
- Search posts by title 
- Paging to load posts step by step
- Login guard on edit and delete routes 

## Learning Outcomes
By completing this project, you will gain hands-on skill in CRUD design, paging logic, search logic, route guard setup, and complete stack web app building using React and Node.js.
